# Склонируй репу, перейди в директорию и запусти контейнеры:
```
git clone git@github.com:DaniilPodovinnikov/lakehouse_in_docker.git

cd lakehouse_in_docker

source start.sh
```

Trino Web UI: http://localhost:8080

MinIO UI: http://localhost:9001 (minio/minio123)

Polaris console: http://localhost:8181 (admin/admin)
